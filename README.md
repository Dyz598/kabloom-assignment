# Kabloom Assignment

### Requirements

- PHP 8.2
- PostgreSQL
- Midtrans for Payment
- Google OAuth for Social Login

### How To Run For Development

For Backend:
```
php artisan serve
```

For Frontend:
```
npm run dev
```
